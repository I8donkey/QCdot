# QBlock - 可视化编程编辑器

![Static Badge](https://img.shields.io/badge/License-MIT-orange)

[English](README.md)

## 概述

QBlock 是一个基于 Qt6 框架的强大可视化编程编辑器。它提供基于节点的界面，通过可视化连接创建程序，非常适合快速原型开发、教学和可视化脚本编写。

## 功能特点

- **可视化节点编程**：通过连接代表函数、变量和 UI 组件的节点来创建程序
- **拖放界面**：从分类面板轻松创建节点
- **实时数据流执行**：以数据流或信号模式运行程序
- **Qt 控件支持**：内置用于创建 Qt6 GUI 应用程序的节点
- **双语界面**：完整支持中文和英文界面
- **主题支持**：深色和浅色主题，舒适的编辑体验
- **保存/加载**：将图形保存为 `.qcd` 文件并稍后重新加载
- **代码导出**：将可视化程序导出为 C++ 源代码

## 安装

### 前置要求

- Qt 6.5 或更高版本
- CMake 3.20 或更高版本
- C++17 兼容编译器（MSVC、MinGW、GCC）

### 从源码构建

```bash
# 克隆仓库
git clone https://github.com/yourusername/qblock.git
cd qblock

# 创建构建目录
mkdir build && cd build

# 使用 CMake 配置
cmake .. -G "MinGW Makefiles"  # 或使用 "Ninja" 加快构建速度

# 编译
cmake --build . --config Release

# 运行编辑器
./examples/QCdot-editor/QCdot-editor.exe
```

## 快速开始

1. **启动编辑器**：运行 `QCdot-editor`
2. **创建节点**：从左侧面板拖动节点或在空白画布上双击
3. **连接节点**：从输出端口单击并拖动到输入端口
4. **编辑节点值**：双击常量节点以更改其值
5. **执行**：按播放按钮以数据流或信号模式运行

## 节点分类

- **输入**：常量（整数、浮点数、布尔、字符串）、变量、计数器
- **输出**：打印节点
- **容器**：主窗口、标签页、布局
- **控件**：按钮、标签、文本框、滑块、复选框、下拉框、数值框、进度条
- **控件方法**：调整大小、移动、显示、隐藏、设置启用、设置文本等
- **数学**：加法、减法、乘法、除法、取模、幂、开方
- **逻辑**：等于、大于、小于、与、或、非、条件分支
- **字符串**：拼接、长度、子串
- **转换**：类型转换节点
- **文件**：文件读写操作
- **工具**：延时、随机数等
